Data Details¶

PassengerId: A unique identifier for each passenger in the dataset. It serves as a reference to distinguish one passenger from another.

Survived: This column indicates whether a passenger survived or not. It is a binary column with values 0 or 1, where 0 typically represents "did not survive" and 1 represents "survived."

Pclass: The passenger class or ticket class represents the socio-economic status of the passenger. It is an ordinal categorical variable with values 1, 2, or 3, where 1 is for the highest class (usually representing the upper class), 2 for the middle class, and 3 for the lower class.

Name: The name of the passenger. It is a text column containing the full name of each individual.

Sex: The gender of the passenger. It is a categorical variable with values "male" or "female," representing the sex of each passenger.

Age: The age of the passenger. It is a numerical column representing the age in years. Some values might be missing, denoted as NaN (Not a Number).

SibSp: The number of siblings or spouses traveling with each passenger. It is a numerical column representing the count of siblings (brothers or sisters) and spouses (husband or wife) aboard the ship.

Parch: The number of parents or children traveling with each passenger. It is a numerical column representing the count of parents and children aboard the ship.

Ticket: The ticket number of the passenger. It is an alphanumeric identifier for each passenger's ticket.

Fare: The fare or price paid by the passenger for the ticket. It is a numerical column representing the monetary cost of the ticket.

Cabin: The cabin number where the passenger stayed. It is an alphanumeric identifier for the cabin.

Embarked: The port of embarkation or the port where the passenger boarded the ship. It is a categorical variable with values "C" (Cherbourg), "Q" (Queenstown), or "S" (Southampton).
